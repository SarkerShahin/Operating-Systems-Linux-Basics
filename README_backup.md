# EXERCISE 1: Linux Mint Virtual Machine

## Objective
Create a Linux Mint Virtual Machine and check the following:
- Distribution details
- Package manager (yum, apt, apt-get)
- CLI editors (Nano, Vi, Vim)
- Software center/software manager
- Default shell for the user

## System Information Checked

### Distribution
- Linux Mint version and codename

### Package Managers
- `apt` - Advanced Package Tool
- `apt-get` - Traditional package manager interface
- `yum` - Not installed (Debian-based systems use apt)

### CLI Editors
- `nano` - Simple and user-friendly editor
- `vi` - Visual editor (original)
- `vim` - Vi Improved

### Software Center
- `mintinstall` - Linux Mint Software Manager

### Default Shell
- `bash` - Bourne Again SHell

# Make the script executable
